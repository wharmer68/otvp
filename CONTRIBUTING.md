# Contributing to OTVP

## How to Contribute

### Feedback on the Spec
- Open an [Issue](../../issues) with your feedback
- Label it `spec-feedback`

### Proposing Changes
1. Fork this repository
2. Create a branch (`git checkout -b my-proposal`)
3. Make your changes
4. Open a Pull Request with a clear description

### Building Reference Agents
- Pick a control domain from the spec
- Follow the agent specification in Section 7
- Submit a PR to `agents/reference/`

## Code of Conduct

Be constructive. Be respectful. Focus on making trust better for everyone.
